# 使用 Workflow 和 Agent 实现多种 RAG 方案

## 项目简介

- **多种 `RAG` 方案的实现：** 基于 `workflow` 和  `Agent` 

## 技术选型

| 技术           | 用途       | 优势                                         |
| -------------- | ---------- | -------------------------------------------- |
| **LangChain**  | Agent 构建 | 构建完全自定义的智能体和基于 LLM 的应用程序  |
| **LangGraph**  | 工作流编排 | 专注于对智能体编排至关重要的底层功能         |
| **LlamaIndex** | 检索抽象   | 标准化文档处理流程，易于集成                 |
| **Milvus**     | 向量数据库 | 高性能、支持混合检索、水平扩展               |
| **vLLM**       | 模型推理   | 高吞吐、低延迟的 LLM/Embedding/Reranker 服务 |
| **Pydantic**   | 结构化输出 | 类型安全，LLM 输出可靠解析                   |
